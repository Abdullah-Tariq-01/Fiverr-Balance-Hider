## Features
- Hide all balances
- Hide on earnings page only
- Show fake money values
- Zero flash (runs at `document_start`)

## Install
1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click Load unpacked
4. Select this folder

## Use
Open the extension popup and toggle options.  
Example fake money: `US$185 to US$10,000`
